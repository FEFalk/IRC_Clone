Help

Commands

[] Optional
<> Required
* Requires permissions

  • /join <#chan> [password] - Join a channel
  • /part [#chan] - Part a channel
  • /topic [#chan] <topic> - * Change channel topic
  • /mode [#chan] <newmode> - * Change channel mode
  • /umode [#chan] <user> <newmode> - * Change user channel permissions
  • /kick [#chan] <user> - * Kick a user from a channel
  • /name <newname> - Change name
  • /quit - Logout from the chat

Modes and Permissions

Channel Modes
  • 1 - Private: Channel will not be listed in the search
  • 2 - Moderated: Users without Channel Voice or Operator cannot talk
Channel Permissions
  • 1 - Voice: User can talk in a Moderated chat
  • 2 - Operator: User can change topic, mode, user modes and kick other users
  • 4 - Banned: User is banned from the channel
